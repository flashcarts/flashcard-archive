
Sie können Programme festlegen, welche in MoonShell2 mit bestimmten Endungen verknüpft werden sollen.
Selbst wenn Sie diese Funktion nicht löschen, sollte der Ordner erhalten bleiben.

Der NDS-Dateiname legt fest, welche Datei geöffnet werden soll.
Fügen Sie zB eine Datei namens "ipk.nds" in diesen Ordner ein, so wird MoonShell2 automatisch Dateien mit der Endnung IPK über die Datei "ipk.nds" laden.
Sollten eine Datei "mp3.nds" oder "jpg.nds" anlegen, so wird die festgelegte Applikation die Priorität übernehmen und den internen Player ablösen.

Sollten Sie eine inkomitable Datei festlegen, wird diese Datei ganz normal aufgerufen.
Wird beispielsweise eine Datei "ReinMoon.nds" in "sav.nds" umbenannt und eingefügt, und eine .sav Datei gegeöffnet so wird ReinMoon gestartet, nicht aber die save-Datei.

Es ist leicht, ExtendLink-kompitable Daten zu erstellen (ein simples Beispiel liegt bei).
Für weitere Informationen sehen Sie in die Datei "moonshl2_extlink_for_developer.txt".
Desweiteren wird beschrieben, wie die Homebrew wieder beendet und zu MoonShell2 zurückgekehrt werden kann (dies ist etwas komplizierter).
