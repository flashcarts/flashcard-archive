
Você pode definir ao MoonShell2 extensões de arquivos para serem abertas usando aplicações externas.
Mesmo que você não use esta função, não exclua esta pasta.

O arquivo NDS definirá qual arquivo será aberto.
Por exemplo, se você copiar o arquivo "ipk.nds" para esta pasta, se você abrir um arquivo IPK o "ipk.nds" será automaticamente carregado.
Se você colocar "mp3.nds" ou "jpg.nds" nesta pasta, a função Extended Link terá prioridade sobre o leitor/visualizador interno do MoonShell.

Se você tentar definir um arquivo incompatível para a função Extended Link, o arquivo será aberto normalmente.
Por exemplo, se você renomear o "ReinMoon.nds" para "sav.nds" e colocar nesta pasta, sempre que você abrir um arquivo .sav o ReinMoon será carregado, e não o save.

É facil criar arquivos compatíveis com a função Extended Link (fiz o mais simples possível).
Por favor leia o "moonshl2_extlink_for_developer.txt" (indisponível) para mais detalhes.
Ele também descreve como sair de uma homebrew e voltar para o MoonShell2 (um pouco complicado)
