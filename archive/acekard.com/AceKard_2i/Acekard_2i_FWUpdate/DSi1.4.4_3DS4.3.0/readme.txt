注意！任何时候都请不要在升级您的AK2i之前升级您的主机！以免无法运行AK2i的升级程序！
此固件支持3DS4.3和DSi1.4.4. 请选择对应主机的文件升级。
升级方法：
请根据您的主机选择运行ak2ifw_update_3ds43_dsi144_onDSi_DSXL_3DS.nds或者ak2ifw_update_3ds43_dsi144_onDSL.nds。运行后按照提示操作即可。
1.拷贝对应的文件到TF卡根目录。
2.如果您是在DSL或者老版DS上运行ak2ifw_update_3ds43_dsi144_onDSL.nds的话，请在运行升级程序后弹出卡，再重新插入，然后按下B键，出现提示后再按下Start键运行升级包即可。
3.如果您是在DSi或DSXL或3DS上升级，请在运行ak2ifw_update_3ds43_dsi144_onDSi_DSXL_3DS.nds后，继续按Start键升级即可。
此升级包暂时不支持HWID为44的AK2i。（您可以在进入AK2i菜单后，按下Start键，选择”帮助“查看对应的HWID。HWID为44的卡只存在于2009年2月以前发售的AK2i。）


Attention! Please NEVER UPDATE your console before you update your AK2i! Or you COULD NOT run your AK2i!
This update support 3DS4.3 and DSi1.4.4. Please refer to readme file in the package. 
Please choose update file according to your console. If you're using DSi or DSXL or 3DS please use ak2ifw_update_3ds43_dsi144_onDSi_DSXL_3DS.nds to update. If you're using DSL or DSPhat please use ak2ifw_update_3ds43_dsi144_onDSL.nds to update. 
Just follow the instructions after you run the update file.
1. Copy the correct update file to the root of your MicroSD card.
2. If you're using ak2ifw_update_3ds43_dsi144_onDSL.nds on your DSphat or DSL, please eject your AK2i and reinsert it after the update program is running. Then press B, then press START key to finish update.
3. If you're using ak2ifw_update_3ds43_dsi144_onDSi_DSXL_3DS.nds on your DSi or DSXL or 3DS, please press START key directly to finish update.
This update file DOES NOT support AK2i whose HWID is '44' temporarily. You can find your HWID of AK2i by selecting "help" after pressing START in AK2i menu. AK2i whose HWID is '44' only exsit in the cards sold before Feb. 2009.
